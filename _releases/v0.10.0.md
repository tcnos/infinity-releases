## Features

### [INVOS-79] Company filters + saved filters
You can now filter the company list and kanban (Sourcing / Dealflow / Portfolio) on the columns you actually see — stage, sector, portfolio manager, currency and more — instead of scrolling or relying on search alone. Filters can be saved per user and recalled from the filter bar, and an existing saved filter can be updated in place rather than recreated. Makes large books navigable and lets each team member keep their own working views.

### [INVOS-70] Labels for Companies & Investors
Funds can now create their own colour-coded labels and apply them to companies and investors — e.g. "Hot", "Follow-up", "Co-invest" — beyond the fixed stage/status fields. Labels are managed in Fund Settings or created on the fly while tagging a record, and show up across lists, kanban cards and profiles (collapsing to colour dots with tooltips in dense views). Each fund keeps its own independent label sets.

### [INVOS-28] @mention members in discussions
Type `@` in a discussion post or reply to mention a fund member from an inline picker. The mentioned person gets an in-app bell notification ("Mention in Discussion in *Company*") that, when clicked, jumps straight to that Fund → Company → Discussions → thread, Slack-style. Mentions fire on both new posts and edits (only newly-added mentions are notified), and you're never notified for mentioning yourself. This release also fixes the notification bell's unread badge, which previously didn't reflect unread counts.

## Enhancements

### [INVOS-68] Create folders from the file picker
When attaching from Drive in discussions, you can now create a new folder directly inside the picker (with a sensible default name) instead of leaving to set it up first. Discussion attachments are also now limited to files only — folders are no longer selectable as attachments — so links always point at something openable.

```bash
docker compose exec inf-backend sh -c "cd /app && npx knex migrate:latest"
docker compose restart inf-backend
```

- `021` — fund / company / investor labels (INVOS-70)
- `022` — notification deep-link column (INVOS-28)
