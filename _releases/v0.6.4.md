## Fix

Portfolio Managers were blank on all companies in any deploy without `IAN_API_KEY` (OrbStack, offline) because the backfill silently skipped when the IAN API was unreachable.

`fund_company_team` is now populated from locally staged `teams` + `users` (pulled by `sync-from-remote`) — no API key, no VPN, no external call required.

## Upgrade steps (OrbStack)

After deploying v0.6.4 images, re-run the migration inside the container:

```bash
docker compose -p invos exec inf-backend \
  node tools/migrate-mongo-to-pg.js --only=fund_company_team
```

> `teams` and `users` must already be in local mongo. If not, run `sync-from-remote` first.
