## Fixes

### Portfolio Manager deduplication
- `backfillPortfolioManagers` now picks the **first** matched PM per company — inserts exactly 1 `fund_company_team` row per company
- Clears existing PM rows before re-inserting so stale duplicates from previous runs don't accumulate
- `--only=fund_company_team` now pre-loads `idMap.companies` from PG so it works in isolation

### Dashboard company counts
- **`getCompanyIdsForFund`** now filters to `display=true` + companies with **≥1 round** — stubs with no investment activity no longer inflate the total
- Result: Active + Past = Total (192 + 53 = 245) — internally consistent and comparable to module views

## Upgrade on OrbStack
```bash
# 1. Deploy new images
docker compose -p invos pull && docker compose -p invos up -d

# 2. Re-run PM backfill (clears duplicates, inserts 1 PM per company)
docker compose -p invos exec inf-backend \
  node tools/migrate-mongo-to-pg.js --only=fund_company_team
```
