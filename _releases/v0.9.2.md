Small UI fix for folder sharing.

## Fixes

### [INVOS-89] "All companies" folder share now shows its label
When a fund folder was shared with **All companies**, that entry appeared as a blank row in the Share dialog's "Currently shared with" list (specific-company shares were unaffected). The row now displays **🌐 All companies** so it's clear who the folder is shared with.

---

## Deployment notes

Frontend-only fix — **no database migration required**. Pull the new images and recreate:
```bash
docker compose pull && docker compose up -d
```
