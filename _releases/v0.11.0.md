## Features

### [INVOS-81] Company Profile enhancements — profile, deal & team fields
Companies now carry a richer profile: Company Stage, Sector, Offering, Business Model and Revenue Model (all backed by editable managed lists), plus structured deal-team roles (Deal Originator / Partner / Manager), founder details (gender, phone, LinkedIn) and social accounts (LinkedIn, Twitter/X, Facebook, Crunchbase, and more). This gives fund managers a far more complete, structured view of each company without resorting to free-text notes.

### [INVOS-47] Portfolio Manager assignment notification
When a user is assigned as the Portfolio Manager of a company, they now receive an in-app bell notification with a deep link to that company. Previously assignments happened silently and the assignee had no way of knowing they were now responsible for a company.

## Enhancements

### [INVOS-93] New profile fields on the public Application Form
The Application Form designer and public form now expose the new company-profile fields (Sector, Company Stage, Offering, Business/Revenue Model, etc.), so applicant-submitted data flows straight into the structured profile instead of being re-keyed by the team.

### [INVOS-94] Backfill of Company Profile fields from legacy data
Existing companies are no longer blank on the new profile fields — the migration tooling now backfills Company Stage, Sector, Offering, Business/Revenue Model, deal team, socials and founder details from the legacy company master. (Coverage reflects what the legacy data actually holds; companies never tagged historically remain blank and can be filled in the UI.)

### [INVOS-92] Managed Lists grouped into category tabs
The Admin → Managed Lists screen now groups lists into category tabs (Company, Investor, etc.) instead of one long flat list, making it much faster to find and edit the right list.

### [INVOS-90] Registered Name in Company Quick Search
Company Quick Search now shows each company's Registered (legal) Name alongside its brand name, so you can tell apart companies that trade under similar names.

---

## Fixes

### [INVOS-91] Fund slug lost on breadcrumbs with a query string
Breadcrumb links that carried a query string were dropping the fund slug, breaking navigation back up the hierarchy. The slug is now preserved regardless of query parameters.

### [INVOS-60] Deleting an in-use currency
It was possible to delete a currency still referenced by funds, companies or rounds, orphaning those records. Deletion is now blocked with a clear message when the currency is in use.

### [INVOS-46] Empty discussion posts
Discussion posts containing only whitespace (or empty rich-text markup) could be submitted, creating blank entries in a thread. Empty and whitespace-only posts are now rejected.
